# Брошура Аптека Атифарм — финален дизайн (Премиум зелено)

Това е **одобреният дизайн от Фаза 1** (виж [`ATIFARM_BROCHURE_SPEC.md`](ATIFARM_BROCHURE_SPEC.md)).
**Data-driven темплейт** — сменяш само данните, а цените, процентите, дисклеймърите и
страниците се генерират автоматично.

## Какво има в пакета
- [`brochure-template.html`](brochure-template.html) — целият дизайн + логика в един файл
  (CSS в `<style>`, данни + рендер в `<script>`). Самостоятелен е.
- `assets/base.css` — механиката на A4 страницата (размери, печат, bleed, break-inside).
- `assets/logo.png` — логото (прозрачно PNG). `assets/logo.pdf` — векторът (за §8).
- `products/*.png` — нормализирани продуктови снимки (бял фон, 700×700).

## Как се ползва веднага (печат / PDF)
Отвори `brochure-template.html` в браузър → бутон **„🖨 Печат / Запази PDF"** (или Ctrl/Cmd+P).
Чекбоксът **„3 мм bleed"** добавя периметър за печатница.

## Какво се редактира всеки брой
Само три неща в `<script>` най-горе:
- `ISSUE` — номер на брой, период, слоган, контакти.
- `FEATURED` — продуктът на корицата (само `oldEur` + `newEur` в евро).
- `PRODUCTS` — масив с продуктите. За всеки:
  - цени: `oldEur`, `newEur` (евро) — левовете и % се смятат автоматично (курс 1.95583),
  - или само отстъпка: `percentOnly:true, percent:15`,
  - `type`: `supplement` | `otc_drug` | `cosmetic` | `medical_device` | `other` → авто-дисклеймър
    (само за `supplement` и `otc_drug`; останалите — без дисклеймър, по §7).

Мрежата се пагинира сама: 6 карти на страница, контактна лента отдолу, чисти page-breaks.

## Прехвърляне в Next.js (Фаза 2)
Темплейтът се вгражда 1:1 като React компоненти: `<BrochureCover>`, `<ProductPage>`,
`<ProductCard>` (режим `percentOnly`), `<SlimFooter>`. Логиката за цени/проценти/дисклеймъри
(`lev`, `bg`, `eurStr`, `levStr`, `pctOf`, `DISCLAIMERS`, `PER_PAGE`) се пренася 1:1 като helpers.
Печат: `@media print` + `print-color-adjust: exact`, A4, клас за 3 мм bleed. Текстове на
български; имена на променливи на английски. **Без нищо извън спецификацията** (§10).
